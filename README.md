# Rocketseat - Product Card

This is a solution to the #boraCodar Product Card challenge by Rocketseat.

## The Challenge

Users should be able to:
- View the optimal layout for the app depending on their device's screen size.
- See an animated view of the product when clicking on the button 360ยบ and to stop it when clicking on the X button.
